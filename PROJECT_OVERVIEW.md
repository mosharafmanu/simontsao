# Project Overview

## Project Summary

This project is a custom WordPress theme build for the A/Prof Simon Tsao website.

The site represents a professional medical practice with the positioning:

- A/Prof Simon Tsao
- Doctor | Surgeon | Researcher

The website should communicate medical credibility, trust, professionalism, clarity, and restraint.

## Build Mindset

This is not a page builder project and not an automated HTML-to-WordPress conversion.

The theme is being built manually so the final result has:

- full control over markup and structure
- clean long-term architecture
- minimal production-quality code
- scalable ACF-driven page building

## Reference Source

The static HTML codebase in this project root is the visual and structural reference for the WordPress build.

Important:

- the frontend was originally exported from a builder
- that output has already been manually cleaned
- builder-specific clutter, unnecessary wrappers, duplicate structures, noisy IDs, and messy classes have already been removed
- future work should treat the static HTML as a clean reference, not as something that still needs builder cleanup

## What Future Sessions Should Assume

- The static reference is already cleaned and organized.
- Theme development should stay manual and deliberate.
- The goal is a professional custom theme, not a generic starter theme.
- Architecture decisions should favor simplicity, clarity, and maintainability.
- Avoid adding abstractions unless there is a clear recurring need.

## Content Direction

The site is expected to center around:

- biography / profile content
- specialist expertise
- research positioning
- appointments and patient information
- contact and practice details
- medically credible section layouts

Implementation should remain professional, polished, and minimal. Avoid flashy, gimmicky, or trend-driven patterns.
